# Dellusion Entertainment

A premium dark-themed website for a live entertainment company targeting Gen Z subcultures, built with React, TypeScript, and Tailwind CSS featuring a goth-cyberpunk aesthetic with glassmorphism UI elements.

![Dellusion Entertainment](https://images.unsplash.com/photo-1516450360452-9312f5e86fc7?w=1200&h=400&fit=crop)

## 🎭 Overview

Dellusion Entertainment is a cutting-edge entertainment platform showcasing live events with a focus on electronic music and immersive experiences. The website features a distinctive dark aesthetic combining cyberpunk UI elements with smooth Motion-powered animations.

## ✨ Features

- **Dark Glassmorphism Aesthetic** - Premium goth-cyberpunk design with HUD-style panels
- **Dual-Accent Color Palette** - Neon magenta (#ff2d9b) and electric cyan (#00e5ff) accents
- **Mobile-First Responsive Design** - Fluid typography using `clamp()` for perfect scaling
- **Motion-Powered Animations** - Spring-physics hover effects on all interactive elements
- **Google Sheets Integration** - Partner and Contact forms with real-time webhook submissions
- **Multi-Page Navigation** - React Router with smooth page transitions
- **Event Showcases** - BANKAI BEATS and DARKCORE event pages

## 🎨 Design System

### Color Palette
- **Primary**: Neon Magenta `#ff2d9b`
- **Secondary**: Electric Cyan `#00e5ff`
- **Bridge**: Deep Purple `#a855f7`
- **Background**: Dark Purple-Black `#08060f`

### Typography
- **Headings**: [Orbitron](https://fonts.google.com/specimen/Orbitron) - Bold, futuristic display font
- **Body**: [Rajdhani](https://fonts.google.com/specimen/Rajdhani) - Clean, technical sans-serif

### UI Elements
- HUD-style panels with corner bracket decorations
- Glassmorphism cards with backdrop blur
- Symmetrical layouts with visual density
- Fast, smooth spring-physics animations

## 🚀 Tech Stack

- **Framework**: React 18 with TypeScript
- **Styling**: Tailwind CSS v4
- **Animations**: Motion (Framer Motion)
- **Routing**: React Router v6
- **Form Handling**: React Hook Form
- **Icons**: Lucide React
- **Build Tool**: Vite

## 🛠️ Installation

### Prerequisites
- Node.js 18+ and pnpm (or npm/yarn)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/atlanoexe/dellusion-entertainment.git
   cd dellusion-entertainment
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   # or
   npm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   # or
   npm run dev
   ```

4. **Open in browser**
   Navigate to `http://localhost:5173`

## 📄 Pages

- **Home**: Landing page with hero section and event highlights
- **About**: Company information and mission statement
- **BANKAI BEATS**: Electronic music festival page (redirects to external site)
- **DARKCORE**: Dark-themed electronic music event showcase
- **Partner**: Partnership inquiry form with Google Sheets integration
- **Past Events**: Gallery of previous events and performances
- **Contact**: Contact form with Google Sheets integration

## 🚢 Deployment

### Build for Production
```bash
pnpm build
# or
npm run build
```

The build output will be in the `dist/` directory.
